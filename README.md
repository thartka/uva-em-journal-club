# UVA Emergency Medicine Journal Club

A web-based platform for the University of Virginia Department of Emergency Medicine Journal Club, featuring interactive visualizations and educational tools.

## Features

- Interactive visualizations for statistical concepts
- Educational tools for understanding statistical distributions
- Mean and standard deviation interpretation exercises

## Project Structure

```
uva-em-journal-club/
├── Apps/              # Interactive applications
├── src/              # Source components
├── static/           # Static assets (images, etc.)
├── styles/           # CSS stylesheets
├── index.html        # Main entry point
└── visualizations.html
```

## Getting Started

Simply open `index.html` in a web browser to view the journal club homepage.
