ROC and AUC

Learn how ROC curves and AUC describe classifier performance using a fixed sepsis prediction dataset (1000 patients).

Page 1: Confusion Matrix

Explore TP, FP, TN, FN and sensitivity, specificity, PPV, NPV as you change the probability threshold.

Page 2: FPR and FNR vs Threshold

See how false positive rate and false negative rate vary with the chosen threshold.

Page 3: ROC Curve and Table

Table of five thresholds with FPR, TPR, sensitivity, specificity and the full ROC curve with five points highlighted.

Page 4: ROC with Threshold Slider

Move the threshold and see the corresponding point on the ROC curve and the metrics.

Page 5: AUC

Area under the ROC curve: a single number that does not depend on the threshold.
