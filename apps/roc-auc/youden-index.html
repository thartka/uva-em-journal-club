6: Youden Index

← Back to Visualizations

The Youden Index J summarizes overall test performance using a single ROC point. It is defined as J = Sensitivity + Specificity − 1.

This point is exactly the one that both maximizes the sum of sensitivity and specificity and is closest to the ideal top-left corner of the ROC space (FPR = 0, TPR = 1).
